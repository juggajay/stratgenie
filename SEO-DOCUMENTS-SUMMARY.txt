INTERNAL LINKING STRATEGY
--------------------------

5-TIER ARCHITECTURE:

Tier 1: Pillar Page Linking
  -> Links OUT to all sub-cluster articles (5-8 links)
  -> Links IN from all sub-articles

Tier 2: Sub-Cluster Article Linking
  -> Links BACK to pillar (required, first reference)
  -> Links BETWEEN related articles (1-2 contextual)
  -> Links to relevant tools (1 max)
  -> Links to cross-silo (1 max, if natural)
  -> Total: 3-5 outbound links per 1,500 words

Tier 3: Tool Page Linking
  -> Links to pillar (1 link)
  -> Links to detailed articles (2-3 links)
  -> Receives 10-15+ quality inbound links

Tier 4: Category/Hub Linking
  -> Links to pillar guides (2-3)
  -> Links to tools (1-2)
  -> Links to related categories (3 max)

Tier 5: Cross-Silo Linking
  -> Max 1-2 links per page
  -> Contextual and natural only
  -> Supports user journey (problem -> solution)

ANCHOR TEXT BREAKDOWN:
  60% Descriptive: "Learn about AGM requirements"
  30% Phrase Match: "NSW strata compliance guide"
  10% Exact Match: "strata compliance" (sparingly)

HEADER HIERARCHY RULES
----------------------

GOLDEN RULE: One H1, Sequential H2->H3->H4

WRONG:
  <h1>Title 1</h1>
  <h1>Title 2</h1>          (Multiple H1s)
  <h3>Subsection</h3>       (Skipped H2)

CORRECT:
  <h1>Page Main Topic</h1>
  <h2>Main Section</h2>
  <h3>Subsection</h3>
  <h3>Another Subsection</h3>
  <h2>Another Main Section</h2>

BY PAGE TYPE:

Blog Post:
  H1: Title
  - H2: Sections
    - H3: Points, FAQ

Pillar Guide:
  H1: Topic
  - H2: Deep sections
    - H3: Sub-topics

Tool Page:
  H1: Tool name
  - H2: How to use
    - H3: Steps

Landing Page:
  H1: Category
  - H2: Topic clusters
    - H3: Items

PRE-PUBLISH CHECKLIST
---------------------

Before publishing any article:

REQUIRED:
  [ ] Title (50-60 chars, includes keyword)
  [ ] Meta description (under 160 chars)
  [ ] One H1, sequential headers
  [ ] Category & pillar/silo assignment
  [ ] Link to pillar page (first reference)
  [ ] 2-3 related article links
  [ ] Image with alt text
  [ ] 3-6 relevant tags
  [ ] Natural keyword usage

RECOMMENDED:
  [ ] Table of contents (if 2,500+ words)
  [ ] FAQ section with schema
  [ ] Tool link (where relevant)
  [ ] Cross-silo link (max 1, if natural)
  [ ] Reading time estimate
  [ ] Author byline

TECHNICAL:
  [ ] Valid markdown/MDX syntax
  [ ] No broken links
  [ ] Mobile-responsive
  [ ] Accessible alt text
  [ ] Pass SEO audit script
