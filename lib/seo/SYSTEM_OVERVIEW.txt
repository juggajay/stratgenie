KEY FORMULAS AT-A-GLANCE
================================================================================

TITLE TAG FORMULA
  [Primary Keyword] | [Value Prop] | StrataGenie
  Length: 50-60 characters
  Example: "AGM Compliance Calendar | Never Miss Deadlines | StrataGenie"

META DESCRIPTION FORMULA
  [Hook/Problem] [Solution] [Proof/Metric] [CTA]
  Length: 150-160 characters
  Example: "Automate AGM, track Strata Hub deadlines & send reminders. Join 500+ NSW schemes. Start free today."

URL FORMULA
  /[section]/[keyword-rich-slug]
  Rules: lowercase, hyphens, readable, no dates (unless blog)
  Example: "/guides/how-to-run-first-agm-self-managed-strata"

A/B TESTING QUICK START
================================================================================

THE PROCESS
  1. Create hypothesis ("X outperforms Y")
  2. Create variant B
  3. Run for 14 days (50% traffic each)
  4. Collect CTR data
  5. Analyze (need 100+ clicks per variant)
  6. Implement winner if +10% improvement
  7. Test winner vs new challenger

SAMPLE TEST
  Hypothesis: "Urgency + numbers outperform generic description"

  | Version | Impressions | Clicks | CTR  |
  |---------|-------------|--------|------|
  | A       | 5,000       | 250    | 5.0% |
  | B       | 5,100       | 281    | 5.5% |

  Result: B wins (+10% CTR) → Implement

================================================================================
GOLDEN RULES (DO NOT BREAK)
================================================================================

✓ Always include CTA ("Start free," "Learn more," "Try now")
✓ Always include brand name ("StrataGenie" - correct spelling)
✓ Never duplicate titles/descriptions across site
✓ Lead with benefit in first 25 characters
✓ Design for mobile (35-char title, 120-char desc)
✓ Include social proof metric (% accuracy, # of users, etc.)
✓ Stay accurate (no misleading headlines)
✓ Use power words (1-2 emotional triggers per page)
✓ Validate quality (use validator function)
✓ Test continuously (A/B test before declaring "done")

================================================================================
COMMON MISTAKES (AVOID!)
================================================================================

✗ "Home" as title → ✓ Use keywords
✗ "Click here" CTA → ✓ Use specific CTA
✗ Keyword stuffing → ✓ Natural language
✗ Duplicate descriptions → ✓ Unique per page
✗ No CTA in description → ✓ Always include CTA
✗ "Welcome to our site" → ✓ Lead with benefit
✗ Description < 120 chars → ✓ Expand to 150-160
✗ Title < 30 chars → ✓ Expand to 50-60
✗ Emojis everywhere → ✓ Use plain text
✗ No social proof → ✓ Include 1 metric
