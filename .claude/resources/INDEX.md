## Use Case Examples

### "We have 500 blog posts and no idea which to refresh"
**Start here:**
1. `content-refresh-implementation.md` → "Content Audit Template"
2. `content-refresh-implementation.md` → "Decay Detection Worksheet"
3. `content-refresh-quick-reference.md` → "PRIORITY MATRIX"
4. Run audits on top 100 pages → score and prioritize

**Expected timeline:** 1 week setup → 2 weeks audit → Week 3 start refreshes

---

### "We know our content is stale but don't know how to update it"
**Start here:**
1. Identify content type (Blog, How-To, Listicle, etc.)
2. Go to `content-refresh-implementation.md` → Find your "Content Type Update Template"
3. Follow the checklist step by step
4. Use `content-refresh-implementation.md` → "Freshness Signal Checklist"
5. Execute re-publishing workflow in main framework

**Expected timeline:** 2-4 hours per article (depending on scope)

---

### "We want to prove refresh ROI before investing heavily"
**Start here:**
1. Pick 3-5 high-traffic pages with decay signals
2. Use `content-refresh-metrics-guide.md` → Collect baseline metrics
3. Refresh using content type template
4. Monitor using monitoring dashboard (1 month)
5. Calculate ROI using framework in metrics guide
6. Present findings to leadership

**Expected timeline:** 30 days to demonstrate ROI

---

### "We want to understand benchmarks for our specific content type"
**Start here:**
1. Go to `content-refresh-metrics-guide.md` → "Metric Benchmarks by Content Type"
2. Find your content type
3. Review traffic, ranking, engagement benchmarks
4. Review ROI expectations
5. Compare your current content to benchmarks

**Expected timeline:** 15 minutes

## Getting Started Checklist

**Day 1:**
- [ ] Read `content-refresh-quick-reference.md` (10 min)
- [ ] Skim `FRAMEWORK-SUMMARY.md` (10 min)
- [ ] Identify your role in the team

**Day 2-3:**
- [ ] Set up Google Analytics access
- [ ] Export Search Console data
- [ ] Create content inventory spreadsheet (using template)

**Day 4-5:**
- [ ] Select top 20 pages to audit
- [ ] Complete decay detection worksheet for each
- [ ] Calculate decay scores

**Day 6-7:**
- [ ] Score pages using priority matrix
- [ ] Identify top 5 pages for refresh
- [ ] Schedule refreshes

**Week 2:**
- [ ] Begin first refresh using content type template
- [ ] Set up metrics tracking
- [ ] Complete refresh and publish

**Week 3+:**
- [ ] Monitor first refresh
- [ ] Begin second refresh
- [ ] Establish monthly cadence
