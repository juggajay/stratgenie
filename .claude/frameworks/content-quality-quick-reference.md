# Content Quality Quick Reference Guide
## One-Page Cheat Sheet for Daily Use

**Last Updated:** 2025-12-04

---

## Publishing Standards Checklist

Before publishing ANY content, verify these minimums:

### Must-Have Requirements
- [ ] Overall quality score: **7.0+** (8.0+ for pillar content)
- [ ] E-E-A-T score: **7.0+**
- [ ] Flesch Reading Ease: **55+**
- [ ] Author bio with credentials
- [ ] Published/updated date visible
- [ ] Primary keyword in title, H1, intro, conclusion
- [ ] 3-5 internal links
- [ ] 2-3 authoritative external links
- [ ] All images have alt text
- [ ] Legal disclaimer (if compliance/legal topic)
- [ ] Mobile responsive
- [ ] No spelling/grammar errors

---

## Content Scoring System

### Overall Score = (Value × 25%) + (SEO × 20%) + (Readability × 20%) + (E-E-A-T × 20%) + (Freshness × 10%) + (Engagement × 5%)

| Score | Rating | Action |
|-------|--------|--------|
| 9.0-10.0 | Exceptional | Publish immediately, promote heavily |
| 8.0-8.9 | Excellent | Publish and promote |
| 7.0-7.9 | Good | Publish, monitor performance |
| 6.0-6.9 | Fair | DO NOT PUBLISH - improve first |
| <6.0 | Poor | Major revision or restart |

---

## E-E-A-T Quick Checklist

### Experience (First-Hand)
- [ ] "We," "our experience," "I've found" language
- [ ] Specific real example (not generic)
- [ ] Process/workflow shown
- [ ] Lessons learned or mistakes shared

### Expertise (Knowledge)
- [ ] Author credentials stated
- [ ] Professional licenses/certifications
- [ ] Years of experience mentioned
- [ ] Technical depth appropriate

### Authoritativeness (Recognition)
- [ ] 3+ citations to gov/peak body sources
- [ ] Links to NSW Fair Trading/legislation
- [ ] Social proof (testimonials, user count)
- [ ] Awards/recognition displayed

### Trustworthiness (Transparency)
- [ ] All claims sourced
- [ ] Legal disclaimer present
- [ ] Contact info accessible
- [ ] HTTPS/SSL certificate
- [ ] Balanced perspective

**Minimum E-E-A-T Score: 7.0/10**

## Readability Standards

### Target Metrics
- **Flesch Reading Ease:** 60-70 (minimum 55)
- **Grade Level:** 8-10 (maximum 12)
- **Sentence Length:** 15-20 words average
- **Passive Voice:** <10% (max 15%)
- **Paragraph Length:** 2-4 sentences

### Formatting Rules
- One H1 only (title)
- H2 every 300-500 words
- Bullet points for 3+ items
- Visual every 300-500 words
- Table of contents if >1500 words
- Bold key phrases (not entire sentences)

### Mobile Requirements
- Minimum 16px font
- Line height 1.5-1.8
- No horizontal scroll
- Touch targets 48×48px minimum

---

## Technical SEO Essentials

### On-Page Elements
- [ ] Primary keyword in title (first 60 chars)
- [ ] One H1 with primary keyword
- [ ] Secondary keywords in H2/H3
- [ ] Keyword density 0.5-1.5%
- [ ] Meta title 50-60 characters
- [ ] Meta description 150-160 characters
- [ ] Clean URL with keywords
- [ ] Schema markup (Article, FAQ, Breadcrumb)
- [ ] Canonical tag set

### Link Requirements
- **Internal:** 3-5 relevant links, descriptive anchors
- **External:** 2-3 authoritative sources
- **No 404s:** All links functional

### Image Optimization
- Descriptive alt text (includes keyword naturally)
- Optimized file size (<200KB)
- One visual per 300-500 words

## Content Length Guidelines

| Content Type | Target Length | Minimum | Maximum |
|--------------|--------------|---------|---------|
| Pillar Content | 3,000-5,000 | 2,500 | 7,000 |
| Cluster Content | 1,500-2,500 | 1,200 | 3,500 |
| How-To Guides | 1,500-3,000 | 1,000 | 4,000 |
| Listicles | 1,000-2,000 | 800 | 3,000 |
| FAQs | 800-1,500 | 500 | 2,000 |

**Rule:** Match or exceed top 3 competitors by 20-30%

## Quality Gates

### Gate 1: Pre-Writing
- [ ] Content brief approved
- [ ] Keywords researched (primary + 3-5 secondary)
- [ ] Top 5 competitors analyzed
- [ ] Outline with H2/H3 structure

### Gate 2: First Draft
- [ ] Word count target met (±20%)
- [ ] Flesch score 55+
- [ ] E-E-A-T signals present
- [ ] Overall score 6.0+

### Gate 3: Editorial
- [ ] Facts verified, sources cited
- [ ] E-E-A-T score 7.0+
- [ ] Technical SEO complete
- [ ] Legal disclaimers added

### Gate 4: Final Approval
- [ ] Overall score 7.0+ (8.0+ pillar)
- [ ] All checklists 100% complete
- [ ] Sign-off from auditor, editor, SEO optimizer, director

### Gate 5: Post-Publishing (2 hours)
- [ ] Page loads correctly
- [ ] No errors (console, GSC)
- [ ] All links functional
- [ ] Schema validates
- [ ] Mobile preview perfect

## Performance Benchmarks

### Traffic & Rankings
- **Traffic growth:** +10% month-over-month
- **Primary keyword:** Top 10 (page 1)
- **CTR:** 3-5% (position-dependent)
- **Impressions:** +15% month-over-month

### Engagement
- **Time on page:** 3-5 minutes
- **Scroll depth:** 60-70%
- **Bounce rate:** 40-60%
- **Pages/session:** 2-3

### Conversion
- **Trial sign-ups:** 2-4%
- **Lead downloads:** 5-8%
- **Email sign-ups:** 3-5%
- **Contact forms:** 1-2%

### Technical
- **Page load (LCP):** <2.5 seconds
- **FID:** <100ms
- **CLS:** <0.1
- **PageSpeed score:** 80+
- **Mobile usability:** 90+

- NSW legislation change (24 hours)
- Ranking drop >10 positions (3 days)

## Common Fixes by Score Range

### Score 6.0-6.9 (Needs Improvement)
**Quick wins:**
- Add 2-3 specific examples from experience
- Strengthen author bio with credentials
- Add 3-5 citations to authoritative sources
- Break up long paragraphs (2-4 sentences)
- Add FAQ section
- Optimize meta description
- Add internal links

### Score 5.0-5.9 (Below Standard)
**Major work needed:**
- Rewrite intro with clear hook and value prop
- Add case study or detailed example
- Increase depth (compare to top 3 competitors)
- Add visuals (images, tables, infographics)
- Complete technical SEO checklist
- Get expert review for accuracy
- Add legal disclaimers

### Score <5.0 (Poor)
**Consider:**
- Complete rewrite
- Different angle/approach
- Abandon topic if no competitive advantage

## Red Flags (Content Killers)

### Do Not Publish If:
- ❌ No author credentials or expertise demonstrated
- ❌ No sources cited for factual claims
- ❌ Outdated information (pre-2024)
- ❌ Spelling/grammar errors throughout
- ❌ No clear value proposition
- ❌ Duplicate/rehashed content
- ❌ Missing legal disclaimers (legal/compliance topics)
- ❌ Broken links or images
- ❌ Not mobile-responsive
- ❌ Page load speed >4 seconds

## One-Minute Self-Audit

**Before you click publish, answer these 10 questions:**

1. [ ] Can I explain in one sentence what unique value this provides?
2. [ ] Have I included a specific example from real experience?
3. [ ] Are my credentials/expertise clearly stated?
4. [ ] Have I cited 3+ authoritative sources?
5. [ ] Can someone scan headings and understand the content?
6. [ ] Is my intro compelling (would I keep reading)?
7. [ ] Have I included 3-5 internal links?
8. [ ] Is there a clear CTA?
9. [ ] Would this rank better than the current top 3 results?
10. [ ] Have I checked for spelling/grammar errors?

**If you answered "No" to more than 2 questions, DO NOT PUBLISH YET.**

## Emergency Content Triage

**Content is live and underperforming badly. Quick diagnosis:**

| Symptom | Likely Cause | Quick Fix |
|---------|-------------|-----------|
| High impressions, low CTR | Weak title/meta | Rewrite title/description, test |
| Low impressions | Not ranking | Build backlinks, internal links |
| High bounce rate | Poor intro or wrong intent | Rewrite first 100 words |
| Low time on page | Poor formatting | Add headings, bullets, visuals |
| No conversions | Weak/missing CTA | Add clear CTA in intro, mid, end |
| Ranking dropped | Competitor improved | Enhance depth, add E-E-A-T |
| Not indexed | Technical issue | Check GSC, submit sitemap |

## Tools Quick Reference

### Readability
- **Hemingway Editor:** Grade level, sentence length, passive voice
- **Yoast SEO:** Readability + SEO analysis
- **Grammarly:** Spelling, grammar, clarity

### SEO
- **Google Search Console:** Rankings, CTR, errors
- **Ahrefs/SEMrush:** Keyword research, competitor analysis
- **Screaming Frog:** Technical audit
- **Schema.org Validator:** Schema markup testing

### Performance
- **Google PageSpeed Insights:** Core Web Vitals, speed
- **GTmetrix:** Detailed performance metrics
- **WebAIM Contrast Checker:** Accessibility

### Analysis
- **Google Analytics:** Traffic, engagement, conversions
- **Hotjar:** Heatmaps, user behavior
- **Microsoft Clarity:** Session recordings
