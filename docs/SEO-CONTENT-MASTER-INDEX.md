### Quick Decision Tree

```
What type of content are you creating?

Complete process or tutorial?
  └─> Pattern A: Comprehensive Guide (2,000-3,000 words)
      Hook: Template 1 or 5
      CTA: Framework 1

Explaining/comparing concepts?
  └─> Pattern B: Explainer/Comparison (1,500-2,000 words)
      Hook: Template 3 or 4
      CTA: Framework 3

New law or regulatory change?
  └─> Pattern C: Compliance Update (1,200-1,800 words)
      Hook: Template 2
      CTA: Framework 4

Solving a specific problem?
  └─> Pattern D: Problem-Solution (1,000-1,500 words)
      Hook: Template 1
      CTA: Framework 1 or 2
```

## SEO Optimization Checklist

### Title
- [ ] 50-60 characters
- [ ] Primary keyword near beginning
- [ ] Includes location (NSW) for local SEO
- [ ] Year for time-sensitive content
- [ ] Compelling, not just keyword-stuffed

### Meta Description
- [ ] 150-160 characters
- [ ] Primary keyword included naturally
- [ ] Clear benefit promise
- [ ] Active voice
- [ ] Ends with period (no cutoff)

### Keywords
- [ ] Primary keyword in title
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in at least one H2
- [ ] Primary keyword 3-5 times in body (0.5-1% density)
- [ ] 2-3 secondary keywords in headings
- [ ] LSI keywords used naturally

### Structure
- [ ] Single H1 (title only)
- [ ] 4-8 H2 headings
- [ ] Keywords in headings where natural
- [ ] Logical H2/H3 hierarchy
- [ ] No orphaned sections

### Links
- [ ] 3-8 internal links with descriptive anchors
- [ ] 1-3 external links to authoritative sources
- [ ] All links working (no 404s)
- [ ] Relevant, contextual linking

### Images
- [ ] WebP or optimized JPG
- [ ] Under 100KB file size
- [ ] 1200x630px dimensions
- [ ] Descriptive file names
- [ ] Alt text with keyword

## Writing Quality Checklist

### Style
- [ ] Professional but approachable tone
- [ ] Plain English (jargon explained)
- [ ] Active voice predominates
- [ ] Second person for instructions
- [ ] Confident and authoritative

### Readability
- [ ] Average sentence: 15-20 words
- [ ] Max sentence: 25 words
- [ ] Paragraphs: 2-3 sentences (max 4)
- [ ] Flesch Reading Ease: 60-70
- [ ] Grade level: 8-10

### Formatting
- [ ] Bold for key concepts
- [ ] Bullets for lists
- [ ] Tables for comparisons
- [ ] Callout boxes for important info
- [ ] Code blocks for formulas/references

### Content Quality
- [ ] Answers reader's question fully
- [ ] Actionable advice with clear steps
- [ ] Real examples included
- [ ] Common mistakes addressed
- [ ] FAQ section comprehensive

### Directory Structure
```

```

### Naming Convention
- Lowercase
- Hyphens (not underscores)
- 3-5 words
- Match URL slug
- No year in slug

## Common Mistakes to Avoid

### Writing
- ❌ Long paragraphs (>4 sentences)
- ❌ Unexplained jargon
- ❌ Passive voice overuse
- ❌ No clear next steps

### SEO
- ❌ Keyword stuffing (>1% density)
- ❌ Missing meta description
- ❌ No internal links
- ❌ Generic "click here" anchors

### Structure
- ❌ Multiple H1 tags
- ❌ Skipping heading levels
- ❌ No FAQ section
- ❌ Missing legal disclaimer

### Product Integration
- ❌ Forced product mentions
- ❌ Multiple CTAs
- ❌ Feature list instead of benefits
- ❌ CTA in introduction
