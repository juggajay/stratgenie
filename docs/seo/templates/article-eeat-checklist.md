## 1. EXPERIENCE - Demonstrating First-Hand Knowledge

### Content Requirements
- [ ] **Real-world examples included**
  - [ ] Minimum 1-2 practical examples from actual cases
  - [ ] Examples are specific and detailed (not generic)
  - [ ] Examples are anonymized appropriately

- [ ] **Practical guidance provided**
  - [ ] Step-by-step instructions where applicable
  - [ ] Common pitfalls identified from experience
  - [ ] "Pro tips" or "Expert insights" callout boxes included

- [ ] **Evidence of hands-on knowledge**
  - [ ] Screenshots, templates, or tools referenced
  - [ ] Specific procedural details that only practitioners would know
  - [ ] Troubleshooting section for common issues

### Experience Signals
- [ ] Author has documented experience in this specific topic
- [ ] Article includes "From the Field" or case study sections
- [ ] Practical checklists or worksheets provided

## 2. EXPERTISE - Demonstrating Specialized Knowledge

### Author Credentials
- [ ] **Author is qualified to write on this topic**
  - [ ] Relevant professional license/certification verified
  - [ ] Years of experience in specific topic documented
  - [ ] Author profile complete and linked

- [ ] **Author attribution is prominent**
  - [ ] Author byline at top of article with photo
  - [ ] Author credentials displayed (title, license)
  - [ ] Link to full author profile included

- [ ] **Multiple authors/reviewers (if applicable)**
  - [ ] Primary author assigned and qualified
  - [ ] Technical reviewer assigned (for YMYL content)
  - [ ] Legal reviewer assigned (for compliance content)
  - [ ] All contributors credited with credentials

### Content Depth
- [ ] **Comprehensive coverage**
  - [ ] Topic covered in appropriate depth for level (Beginner/Intermediate/Advanced)
  - [ ] Minimum word count met: Beginner (1,200), Intermediate (1,800), Advanced (2,500+)
  - [ ] All major aspects of topic addressed

- [ ] **Technical accuracy**
  - [ ] All facts verified against primary sources
  - [ ] Legislation references are current and accurate
  - [ ] State-specific variations documented
  - [ ] No contradictions or inaccuracies

- [ ] **Specialized terminology used appropriately**
  - [ ] Industry terms defined on first use
  - [ ] Glossary provided for complex articles
  - [ ] Jargon minimized but not oversimplified

## 3. AUTHORITATIVENESS - Demonstrating Industry Recognition

### Citations & Sources
- [ ] **Primary sources cited**
  - [ ] Minimum 3 primary sources (legislation, government sites)
  - [ ] All legislation references include section numbers and URLs
  - [ ] Official regulatory body sources cited where applicable

- [ ] **Secondary authoritative sources**
  - [ ] Industry association sources (SCA, REI) referenced
  - [ ] Professional publications cited
  - [ ] Academic or research sources included (if relevant)

- [ ] **Source quality verified**
  - [ ] All sources passed source hierarchy check (see framework)
  - [ ] No low-quality or anonymous sources used
  - [ ] Source recency verified (prefer sources < 2 years old)

- [ ] **Citation format consistent**
  - [ ] Inline citations with superscript numbers
  - [ ] Full reference list at end of article
  - [ ] All external links functional and to correct pages
  - [ ] Access dates included for all sources

### Internal Authority Signals
- [ ] **Links to pillar content**
  - [ ] Links to relevant pillar page
  - [ ] Links from related articles added
  - [ ] Minimum 3-5 relevant internal links

- [ ] **Topic cluster integration**
  - [ ] Article assigned to correct topic cluster
  - [ ] Breadcrumb navigation correct
  - [ ] Related articles module populated

### External Authority Signals
- [ ] **Outreach prepared**
  - [ ] List of 5-10 sites that might link to this content
  - [ ] Notification sent to mentioned organizations
  - [ ] Social sharing plan created

## 4. TRUSTWORTHINESS - Demonstrating Reliability & Safety

### Trust Signals
- [ ] **Publication & update dates**
  - [ ] Original publication date displayed
  - [ ] Last updated date displayed
  - [ ] Last regulatory check date included (for compliance content)

- [ ] **Transparency about content**
  - [ ] Purpose of article clearly stated in intro
  - [ ] Disclaimers included (legal, financial, or professional advice)
  - [ ] Limitations acknowledged (jurisdiction-specific, etc.)

- [ ] **Contact information**
  - [ ] Author contact method provided
  - [ ] General support contact visible
  - [ ] Feedback mechanism included

### YMYL-Specific Requirements (If Applicable)

#### Legal Content
- [ ] **Legal disclaimers prominent**
  - [ ] "This is not legal advice" disclaimer visible
  - [ ] Jurisdiction clearly specified
  - [ ] Recommendation to consult professional included

- [ ] **Regulatory accuracy**
  - [ ] All legislation current as of publication date
  - [ ] State/territory variations documented
  - [ ] Links to official legal sources included

- [ ] **Legal review completed**
  - [ ] Legal reviewer assigned and credited
  - [ ] Review sign-off obtained
  - [ ] Review date documented

#### Financial Content
- [ ] **Financial disclaimers**
  - [ ] "This is not financial advice" disclaimer visible
  - [ ] Recommendation to consult accountant/financial advisor included

- [ ] **Financial accuracy**
  - [ ] All calculations verified
  - [ ] Tax implications noted where relevant
  - [ ] Currency and dates specified

- [ ] **Financial expert review**
  - [ ] CPA or financial professional reviewed
  - [ ] Review sign-off obtained

#### Compliance Content
- [ ] **Compliance disclaimers**
  - [ ] State-specific nature of compliance noted
  - [ ] Recommendation to verify current requirements included

- [ ] **Regulatory currency**
  - [ ] Last checked against current regulations
  - [ ] Next review date scheduled
  - [ ] Changes from previous versions noted (if update)

### Content Quality
- [ ] **Editorial quality**
  - [ ] Spell-checked and grammar-checked
  - [ ] Reviewed by editor
  - [ ] Formatted consistently with style guide
  - [ ] No broken formatting or display issues

- [ ] **Visual quality**
  - [ ] Images are high-quality and relevant
  - [ ] All images have alt text
  - [ ] Charts/graphs properly labeled
  - [ ] Brand visual guidelines followed

- [ ] **Accessibility**
  - [ ] Heading hierarchy correct (H1 → H2 → H3)
  - [ ] Link text is descriptive (not "click here")
  - [ ] Color contrast sufficient
  - [ ] Content readable at Grade 8-10 level (for beginner content)

## 5. Technical SEO & Structure

### On-Page SEO
- [ ] **Meta information optimized**
  - [ ] Title tag includes target keyword (50-60 characters)
  - [ ] Meta description compelling and accurate (150-160 characters)
  - [ ] URL slug is clean and descriptive
  - [ ] Featured image selected and optimized

- [ ] **Content structure**
  - [ ] H1 tag used once (article title)
  - [ ] H2 and H3 tags used logically for sections
  - [ ] Paragraphs are scannable (3-4 sentences max)
  - [ ] Bullet points and lists used appropriately

- [ ] **Keyword optimization**
  - [ ] Primary keyword in: Title, H1, first paragraph, meta description
  - [ ] Secondary keywords naturally integrated
  - [ ] Not over-optimized or keyword-stuffed
  - [ ] LSI keywords included

### Schema Markup
- [ ] **Article schema implemented**
  - [ ] Article type specified
  - [ ] Author information complete
  - [ ] Publication dates included
  - [ ] Publisher information included

- [ ] **Additional schema (if applicable)**
  - [ ] FAQPage schema for Q&A sections
  - [ ] HowTo schema for step-by-step guides
  - [ ] ReviewedBy schema for reviewed content

- [ ] **Schema validated**
  - [ ] Tested in Google Rich Results Test
  - [ ] No errors or warnings
  - [ ] Preview looks correct

### Performance
- [ ] **Page speed optimized**
  - [ ] Images compressed (< 200KB each)
  - [ ] Lazy loading implemented
  - [ ] No render-blocking resources

- [ ] **Mobile optimization**
  - [ ] Mobile preview checked
  - [ ] Content readable on small screens
  - [ ] Buttons/links easily tappable

## 7. Post-Publication Requirements

### Immediate Actions
- [ ] **Quality checks**
  - [ ] Live page reviewed on desktop
  - [ ] Live page reviewed on mobile
  - [ ] All links tested and working
  - [ ] Schema markup validated on live page

### Promotion
- [ ] **Internal promotion**
  - [ ] Linked from relevant existing articles
  - [ ] Added to newsletter queue
  - [ ] Shared on company social media

- [ ] **External promotion**
  - [ ] Outreach emails sent to potential linkers

### Monitoring
- [ ] **Tracking setup**

  - [ ] Rank tracking keywords added

- [ ] **Maintenance scheduled**
