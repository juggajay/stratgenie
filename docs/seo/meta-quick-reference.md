## The 30-Second Framework

### Title Formula
```
[Primary Keyword] | [Value Prop] | StrataGenie
```
**Example:** `AGM Compliance Calendar | Never Miss Deadlines | StrataGenie` (58 chars âœ“)

**Rules:**
- 50-60 characters max
- Brand name last
- Max 2 keywords
- Title case

### Description Formula
```
[Hook/Problem] [Solution] [Proof/Metric] [CTA]
```
**Example:** `Automate AGM compliance, track Strata Hub deadlines & send reminders. Join 500+ NSW schemes. Start free today.` (157 chars âœ“)

**Rules:**
- 150-160 characters
- Start with benefit/hook
- Include 1 social proof metric
- End with CTA
- Active voice

### URL Formula
```
/[section]/[keyword-rich-slug]
```
**Example:** `/guides/how-to-run-agm-self-managed-strata` (lowercase, hyphens only)

**Rules:**
- Lowercase only
- Hyphens for word breaks
- Avoid dates (unless blog)
- Under 75 characters
- Human-readable

## Emotional Triggers Quick Sheet

### Pain Avoidance (Fear)
"Avoid [penalty]. StrataGenie [solution]."
*Example:* "Avoid $1,000 penalties. StrataGenie tracks every deadline."

### Aspiration (Achievement)
"Become [goal]. [Solution] helps you."
*Example:* "Become the most organized committee. StrataGenie tracks everything."

### Social Proof (Belonging)
"Join [number] schemes. [Benefit]."
*Example:* "Join 500+ NSW schemes staying compliant."

### Scarcity (Limited)
"Limited time: [offer]. [Deadline]."
*Example:* "Limited time: 14-day free trial. Start today."

### Relief (Solution)
"Finally, [solution] for [problem]."
*Example:* "Finally, stress-free AGM compliance."

## A/B Testing Rapid-Start

**Test hypothesis:** "X outperforms Y on [page] measured by CTR"

### Week 1-2: Run Test
1. Create variant B
2. Implement on 50% of traffic
3. Run 14 days minimum
4. Collect daily data

### Week 3: Analyze
1. Compare CTR: Variant A vs B
2. Need 100+ clicks per variant for significance
3. 10%+ improvement = winner
4. Document results

### Week 4+: Implement Winner
1. Roll out winner to 100%
2. Plan new challenger (variant C)
3. Start next test
4. Rotate winners vs challengers

## Implementation Command Cheat Sheet

### Generate Title
```typescript
import { titleTemplates } from '@/lib/seo/meta-templates';

const title = titleTemplates.featureBenefitBrand({
  featureName: "AGM Compliance",
  valueProp: "Never Miss Deadlines"
});
// Result: "AGM Compliance | Never Miss Deadlines | StrataGenie"
```

### Generate Description
```typescript
import { descriptionTemplates } from '@/lib/seo/meta-templates';

const desc = descriptionTemplates.problemSolveProof({
  problem: "Struggling with compliance",
  solution: "StrataGenie automates everything",
  socialProof: "500+ NSW schemes",
  cta: "Start free"
});
```

### Validate Metadata
```typescript
import { validateMetadata } from '@/lib/seo/meta-templates';

const result = validateMetadata(title, description);
if (!result.isValid) {
  console.error(result.errors);
}
console.log(`Score: ${result.score}/100`);
```

### Create URL Slug
```typescript
import { slugTemplates } from '@/lib/seo/meta-templates';

const slug = slugTemplates.blogPost(
  "How to Run Your First AGM",
  2025
);
// Result: "how-to-run-your-first-agm-2025"
```

## Monthly Optimization Ritual

**First Monday of month (15 mins):**

1. **Pull GSC Data** (3 mins)
   - Check impressions, clicks, CTR
   - Identify top 5 & bottom 5 pages

2. **Flag Issues** (5 mins)
   - Any missing metadata?
   - Duplicate titles/descriptions?
   - Pages with low CTR?

3. **Plan Tests** (5 mins)
   - Pick 1 bottom 5 page
   - Create variant B
   - Document hypothesis

4. **Document** (2 mins)
   - Add to test tracker
   - Set start/end dates
   - Assign owner
