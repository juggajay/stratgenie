## The Three Rules

1. **Different Keywords = Different Pages** (safe to have both)
2. **Same Keyword = Different Intent** (informational vs. transactional = safe)
3. **Similar Content + Similar Keywords** = Cross-link or consolidate

## Common Scenarios & Solutions

| Problem | Solution | Time |
|---------|----------|------|
| 2 blog posts, same keyword | Redirect weaker to stronger | 5 min |
| Blog + pillar, same keyword | Merge blog into pillar + redirect | 30 min |
| Blog + tool, same keyword | Differentiate by intent (how-to vs. calculator) | 15 min |
| Tool + landing page | Differentiate by audience | 20 min |
| Old + new content | Redirect old if outdated | 10 min |

## FAQ

**Q: Do I need to read all four documents?**
A: No. Start with Quick Start + Audit. Read Framework only for detailed guidance.

**Q: How long does pre-publish check take?**
A: 5-10 minutes per article.

**Q: What if I find cannibalization?**
A: Use Framework or Quick Start to find your scenario, then use appropriate Template.

**Q: Can a page target multiple keywords?**
A: Yes. One PRIMARY keyword (unique per page), but secondary/long-tail can overlap 5-30%.

**Q: Is 80% the magic overlap number?**
A: Yes. 80%+ keyword overlap = consolidate. Below 80% = can differentiate.
